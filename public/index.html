Cómo funciona

  1. El ESP32 mide la distancia con el sensor JSN‑SR04T (promedio de lecturas).
  2. Envía distance_cm por HTTP a /api/reading con su token.
  3. El servidor calcula Ocupado si la distancia ≤ umbral.
  4. La interfaz se actualiza en tiempo real vía Socket.IO.
Consejo: ajusta el THRESHOLD_CM en .env según tu altura de montaje.
